arkouda.security.get_arkouda_client_directory() → str¶

A platform-independent means of finding path to the current user’s .arkouda directory where artifacts such as server access tokens are stored.

The default implementation is to place the .arkouda directory in the current user’s home directory. The default can be overridden by seting the ARKOUDA_HOME environment variable.

Returns

string corresponding to .arkouda directory path

Return type

str
